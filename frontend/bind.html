2

Approve Health Access

Apple protects Health data, so the first run needs one manual approval.

  1. A
    Run the shortcut once

    If this message appears, Shortcuts still needs permission to read Steps.

  2. B
    Tap Allow or Turn On All

    When the Health permission sheet appears, enable every requested category.

  3. C
    Confirm Steps is enabled

    The Steps switch should be green. Then run the shortcut again to sync.

!

No Samples Found

This is the common warning shown before Health permission is granted.

3

Fix Permissions Later

If you tapped Don't Allow before, open Settings and turn Steps back on for Shortcuts.

1

Open Settings, then Health

Go to Settings, scroll to Apps, then choose Health.

2

Open Data Access & Devices

Inside Health, tap Data Access & Devices, then find Shortcuts in the app sources.

3

Allow Shortcuts to read Steps

If the top button says Turn Off All and Steps is green, the setup is correct.

Path

Settings > Apps > Health > Data Access & Devices > Shortcuts > Steps
